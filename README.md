# Freelance Marketplace

## Description

Freelance Marketplace is an Android-based platform designed to connect employers and freshers. The application enables employers to post job opportunities and allows freshers to browse and apply for jobs through a simple and user-friendly interface.

The platform uses Firebase for secure user authentication and real-time database management, ensuring seamless communication between job seekers and recruiters.

## Features

- Employer registration and login
- Fresher registration and login
- Post and manage job listings
- Browse available job opportunities
- Apply for jobs directly through the application
- Secure user authentication using Firebase Authentication
- Real-time data storage and synchronization using Firebase Realtime Database / Firestore
- User-friendly Android interface

## Technologies Used

- Android Studio
- Java / Kotlin
- Firebase Authentication
- Firebase Realtime Database / Cloud Firestore
- XML
- Material Design Components

## Project Highlights
